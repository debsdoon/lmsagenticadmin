## Design System

### Color Palette
- **Primary**: #2563EB (Blue)
- **Secondary**: #7C3AED (Purple)
- **Success**: #10B981 (Green)
- **Warning**: #F59E0B (Amber)
- **Error**: #EF4444 (Red)
- **Background**: #F9FAFB (Light Gray)
- **Text**: #111827 (Dark Gray)

### Typography
- **Headings**: Inter, 600-700 weight
- **Body**: Inter, 400 weight
- **Code**: JetBrains Mono, 400 weight

### Components
- **Buttons**: Rounded corners (8px), clear hierarchy
- **Cards**: Subtle shadow, rounded corners (12px)
- **Inputs**: Clear borders, focus states
- **AI Chat**: Distinct styling with AI avatar
- **Suggestions**: Highlighted with icon and action buttons

### Accessibility
- WCAG 2.1 AA compliance
- Keyboard navigation support
- Screen reader friendly
- High contrast mode support
- Focus indicators

### Responsive Breakpoints
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px
